# isro-project-vadica

## Overview
VADICA is an AI-powered dual intelligence system developed for the ISRO Hackathon 2025. It aims to enable smarter space research and autonomous decision-making.

## Features
- Advanced AI algorithms for space research.
- Autonomous decision-making capabilities.
- Integration with ISRO's existing systems.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/abhishekdeyz/isro-project-vadica.git
   ```
2. Navigate to the project directory:
   ```bash
   cd isro-project-vadica
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
Start the development server:
```bash
npm run dev
```
